# Automate Ocean Protocol with Our New n8n Community Node

We're excited to announce the launch of n8n-nodes-ocean-protocol, a powerful new community node that brings Ocean Protocol's decentralized data marketplace capabilities directly into your n8n workflows.

## The Challenge: Bridging Web3 Data Markets and Automation

As Web3 continues to evolve, Ocean Protocol has emerged as a leading solution for tokenizing data assets and enabling secure data sharing. However, integrating Ocean Protocol's capabilities into automated workflows has traditionally required custom development and deep technical knowledge. Data scientists, DAOs, and Web3 projects need seamless ways to interact with Data NFTs, Datatokens, and Compute-to-Data features without writing complex code.

## Introducing n8n-nodes-ocean-protocol

Velocity BPA has developed this community node to democratize access to Ocean Protocol's powerful features. Whether you're managing Data NFTs, minting Datatokens, executing Compute-to-Data jobs, or tracking veOCEAN governance tokens, you can now do it all through n8n's intuitive visual interface.

## Key Features

- **Data NFT Management**: Create, update, and query Data NFTs without writing a single line of Solidity
- **Datatoken Operations**: Mint, transfer, and manage datatokens programmatically within your workflows
- **Compute-to-Data Integration**: Trigger and monitor compute jobs on private datasets while maintaining data privacy
- **veOCEAN Tracking**: Monitor governance token balances and participate in Ocean Protocol's governance
- **Seamless Authentication**: Built-in wallet integration for secure blockchain interactions

## Real-World Use Cases

This node opens up numerous automation possibilities: automatically mint datatokens when new datasets are published, trigger Compute-to-Data jobs based on external events, send notifications when veOCEAN balances change, or create complete data marketplace workflows that span Web2 and Web3 systems.

## Getting Started

Installation is straightforward. In your n8n instance, simply run:


npm install n8n-nodes-ocean-protocol


Restart n8n, and you'll find the Ocean Protocol node available in your node palette. Check out the [GitHub repository](https://github.com/Velocity-BPA/n8n-nodes-ocean-protocol) for detailed documentation, example workflows, and configuration guides.
